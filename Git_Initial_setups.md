# GIT Setups:

```
mkdir /busapp
cd /busapp
touch file1 file2
cd ..
git init busapp
```

Busapp is our project and git init busapp command used to get in the busapp folder into the VCS(version control system)

If we ran the command it creates a .git dir inside the busapp.
